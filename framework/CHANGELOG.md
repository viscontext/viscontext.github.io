# Framework changelog

## 0.4.0 — Imported source records

- Add provenance for metadata normalized from external visualization catalogs.
- Add eleven attributed records from NASA SVS, Our World in Data, the UK
  Co-Benefits Atlas, and the IPCC.
- Permit explicit null values when a source does not provide a framework field.
- Separate visualizations from representative video, image, frame-set, data,
  metadata, and interactive media assets.
- Expand contributor roles and support affiliations.

## 0.3.0 — Visualization collections

- Add single, complete multi-visualization, ordered-story, and sampled
  large-collection project structures.
- Add visualization-specific encoding and limitation notes.

## 0.2.0 — Experimental catalog fixture

- Replace the single placeholder section with six provisional metadata sections.
- Add six fictional visualization projects for catalog and detail-page testing.
- Add project type, topic, year, and visualization-kind fields for browsing.

## 0.1.0 — Experimental
